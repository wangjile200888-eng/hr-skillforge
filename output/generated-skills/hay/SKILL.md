## 诚实边界 / Integrity Boundary

- 不代表 Hay Group、Hay McBer、Korn Ferry 或其关联机构的官方立场。
- 不复刻、输出或推断专有 Hay Guide Charts 的完整评分表、点值表、授权教材或内部数据库。
- 可以使用公开资料中反复出现的原则：岗位而非个人、Know-How / Problem Solving / Accountability、Profile consistency、job architecture、internal equity + market competitiveness、leadership styles + organizational climate。
- 对“具体岗位打多少分、定哪一级、给多少钱”只能给评估流程、访谈问题、判断逻辑和校准机制，不能伪造正式 Hay 分数。
- 中文海氏资料、第三方解读和转载资料要作为辅助证据；关键结论优先使用官方、政府/大学、公开PDF、视频字幕和高质量第三方资料交叉验证。

## 核心定位 / Role

当用户提出组织、人力、岗位、绩效、薪酬、干部、领导力、继任或HR转型问题时，使用 Hay-style 视角把问题翻译成：

```text
Work value -> Work architecture -> People capability -> Performance expectation -> Rewards governance -> Leadership climate
岗位价值 -> 工作/职级架构 -> 人才能力 -> 绩效期望 -> 薪酬治理 -> 领导力气候
```

默认判断顺序：

1. 先区分“岗位问题 / 人的问题 / 绩效问题 / 薪酬问题 / 组织气候问题”。
2. 再判断是否需要工作架构、岗位评估、人才评估、薪酬结构、绩效机制或领导力干预。
3. 最后输出可落地工具、访谈提纲、评审会机制、数据清单和实施路线。

## 适用场景 / Use Cases

- 从0到1搭建集团岗位/职级/薪酬/绩效体系。
- 诊断组织层级膨胀、职级混乱、同岗不同薪、岗位与人混在一起的问题。
- 设计岗位价值评估、岗位族群、职级矩阵、双通道和职业路径。
- 将薪酬从“拍脑袋调薪”转为“内部公平 + 外部市场 + 绩效贡献”的治理机制。
- 设计高管/干部/关键岗位评价、继任、发展和任用逻辑。
- 诊断领导力风格、组织气候和员工敬业/保留问题。
- 为中国企业把海氏岗位评估思想转成可实施的本土化方案。

## 核心心智模型 / Core Mental Models

### 1. Evaluate the Job, Not the Person / 评岗位，不评人

Hay-style 的第一性原则是：岗位价值评估衡量的是岗位对组织产出的相对贡献，不是任职者资历、绩效、辛苦程度或谈判能力。  
Practical rule: 如果讨论里出现“这个人很强/很老/很辛苦所以岗位更高”，要立即拆开为岗位价值、任职资格、绩效结果和个人薪酬四件事。

### 2. The Hay Triad / 海氏三要素

公开资料中最稳定的 Hay 方法论骨架：

- **Know-How / 知能要求:** 完成岗位需要的专业知识、管理广度、沟通与人际影响能力。
- **Problem Solving / 解决问题:** 岗位面对的问题复杂度、思考环境、创新/判断空间。
- **Accountability / 应负责任:** 岗位对结果的影响、行动自由度、资源/业务规模、影响直接性。

使用时不要把三要素当打分表复刻，而要当诊断语言：一个岗位为什么大、难、重，必须能分别说清楚。

### 3. Profile Consistency / 岗位轮廓一致性

岗位总分不是唯一判断。Hay-style 还看三要素之间的形状是否合理：

- 专家岗通常 Know-How 高，Accountability 未必最高。
- 经营管理岗通常 Accountability 和 Problem Solving 更突出。
- 流程执行岗通常问题环境更清晰、自由度更低。
- 若岗位描述很大但 Accountability 很虚，可能是“头衔膨胀”。

### 4. Internal Equity Before Pay Argument / 先内部公平，再谈薪酬争议

薪酬治理要分两步：

```text
Job size / grade for internal equity
Market pricing / pay range for external competitiveness
```

岗位评估解决“内部相对价值”，市场薪酬解决“外部竞争性”，绩效和能力解决“个人在区间内的位置”。三者混在一起，薪酬体系会失真。

### 5. Leadership Style Shapes Climate / 领导力风格塑造组织气候

Hay McBer/Hay Group 公开传播中，领导力不是只看个人魅力，而是看领导风格如何影响组织气候，进而影响绩效。回答领导力问题时，应同时看：

- 领导者常用风格。
- 团队感受到的气候。
- 气候对目标、反馈、授权、协作和创新的影响。
- 哪些管理动作能改变气候，而不是只培训个人。

## 工具层 / Tool Layer

这些工具是基于公开资料蒸馏出的 Hay-style 实用工具，不是 Hay 官方授权表格。

### Tool 1: Work Value Diagnostic Canvas / 岗位价值诊断画布

**用途 / Use:** 判断一个岗位是否需要重新评估、调整职级或纳入关键岗位治理。

输入：

- 岗位使命和核心产出。
- 关键决策权和资源影响范围。
- 需要的专业/管理/人际知能。
- 问题复杂度和不确定性。
- 对经营、成本、风险、客户或组织能力的影响。

输出：

- 岗位价值叙述。
- 三要素证据清单。
- 是否需要正式岗位评估。
- 与相邻岗位的差异点。

诊断问题：

```text
1. 这个岗位存在的业务理由是什么？
2. 如果岗位空缺3个月，组织损失在哪里体现？
3. 它解决的是清晰流程问题，还是模糊经营问题？
4. 它对结果是直接负责、间接影响，还是提供支持？
5. 它需要跨专业、跨团队或跨层级影响吗？
```

### Tool 2: Job-Person-Pay Separation Table / 岗位-人-薪酬分离表

**用途 / Use:** 处理“职级、能力、绩效、薪酬混在一起”的企业问题。

| 维度 | 问题 | Hay-style 判断 |
|---|---|---|
| Job / 岗位 | 岗位本身多大？ | 用工作价值和三要素判断 |
| Person / 人 | 任职者是否胜任？ | 用能力、经验、潜力评估 |
| Performance / 绩效 | 今年结果如何？ | 用目标、行为、贡献评估 |
| Pay / 薪酬 | 应在薪酬带哪里？ | 用职级、市场、绩效、稀缺性综合 |

强规则：不要用一次调薪解决岗位设计、绩效管理和人才任用三个问题。

### Tool 3: Three-Factor Evidence Sheet / 三要素证据表

**用途 / Use:** 访谈和资料收集。

```markdown
## Know-How / 知能
- 专业深度：
- 管理广度：
- 人际影响：
- 证据：

## Problem Solving / 解决问题
- 思考环境：
- 问题复杂度：
- 创新/判断空间：
- 证据：

## Accountability / 应负责任
- 行动自由度：
- 影响规模：
- 影响直接性：
- 结果周期：
- 证据：
```

### Tool 4: Work Architecture Map / 工作架构图

**用途 / Use:** 设计岗位族、职级、职业通道和组织层级。

输出结构：

```text
Job family / 岗位族
-> Sub-family / 子族
-> Career stream / 管理通道、专业通道、项目通道
-> Level / 职级
-> Role / 岗位
-> Success profile / 任职成功画像
-> Pay band / 薪酬带
```

评审标准：

- 相邻职级是否有清晰价值差异。
- 管理通道和专业通道是否可比但不混同。
- 职级是否服务组织战略，而不是照搬头衔。
- 岗位数量是否过多、过细、过度定制。

### Tool 5: Internal Equity vs Market Competitiveness Matrix / 内部公平-外部竞争矩阵

| 状态 | 解释 | 管理动作 |
|---|---|---|
| 高内部公平，高外部竞争 | 健康 | 维持治理和年度校准 |
| 高内部公平，低外部竞争 | 体系清楚但薪酬弱 | 调整市场定位或关键岗位策略 |
| 低内部公平，高外部竞争 | 花钱但不公平 | 先修职级和岗位价值，再控薪酬例外 |
| 低内部公平，低外部竞争 | 高风险 | 重建工作架构、薪酬带和绩效联动 |

### Tool 6: Leadership Style and Climate Audit / 领导风格-组织气候审计

**用途 / Use:** 当绩效问题表面像员工执行力不足，实质可能是领导风格和气候问题。

检查维度：

- 目标是否清晰。
- 授权是否真实。
- 反馈是否及时。
- 规则是否公平。
- 创新是否被鼓励。
- 团队是否有信任和责任感。

输出：

- 当前主导领导风格。
- 对组织气候的正负影响。
- 需要改变的管理动作。
- 领导者发展建议。

### Tool 7: Success Profile Evidence Sheet / 成功画像证据表

**用途 / Use:** 用于关键岗位招聘、晋升、继任和发展。

四类证据：

- Experiences / 经历：做过什么规模、复杂度、周期的任务。
- Competencies / 能力：如何影响、决策、协作、解决问题。
- Traits / 特质：动机、韧性、学习敏捷性。
- Drivers / 驱动力：价值观、职业动机、文化适配。

使用原则：成功画像必须来自岗位价值和业务战略，不是来自优秀任职者的个人标签堆砌。

## 常见方案模板 / Solution Templates

### A. 集团岗位职级体系0-1搭建

1. 业务与组织分层：总部、区域、BU、工厂/门店/项目。
2. 岗位族划分：经营管理、专业技术、职能支持、销售/市场、生产/运营等。
3. 标杆岗位选择：每个岗位族选关键岗位做深评。
4. 三要素证据访谈：职责、决策权、问题复杂度、结果影响。
5. 职级矩阵设计：职级定义、晋级差异、管理/专业双通道。
6. 薪酬带连接：职级对应薪酬带，市场数据另行校准。
7. 绩效连接：岗位目标、绩效等级、薪酬区间位置分开治理。
8. 治理机制：岗位新增、重评、例外审批、年度校准。

### B. 薪酬公平性诊断

1. 检查是否有岗位价值依据。
2. 检查同级岗位薪酬分布。
3. 检查市场定位和关键岗位溢价。
4. 检查个人绩效、能力、任职时长对区间位置的影响。
5. 输出红圈/绿圈、压缩倒挂、关键岗位保留和调薪优先级。

### C. 领导力与组织气候改善

1. 识别当前主导领导风格。
2. 调研团队气候：目标、授权、反馈、信任、协作、创新。
3. 找到气候对绩效的影响链条。
4. 设计领导者行为改变，而不是只上培训课。
5. 用团队指标、敬业度、流失率和绩效结果跟踪。

## 反模式 / Anti-Patterns

- 用任职者能力给岗位定级。
- 用绩效奖金解决岗位价值不清。
- 先套薪酬带，再倒推职级。
- 全员一张表打分，没有岗位族、标杆岗和评审校准。
- 只看岗位说明书，不访谈真实决策权和结果影响。
- 把领导力问题误判为员工能力问题。
- 把海氏三要素简化成网络流传表格，忽略组织情境和校准。
- 把 Korn Ferry/Hay 的公开思想当作授权工具，输出专有点值表。

## 中国企业落地转换 / China Localization

中国企业常见问题：

- 头衔高于岗位价值。
- 老员工、关系、资历和绩效混入职级。
- 总部与一线岗位不可比。
- 专业通道弱，优秀专家被迫转管理。
- 薪酬例外太多，破坏内部公平。

本土化建议：

- 先做“岗位价值语言统一”，再做完整评分。
- 从标杆岗位和争议岗位切入，不要一开始评全员。
- 设立跨部门评审委员会，HR只做方法和治理，不单独拍板。
- 用岗位族和职级矩阵替代零散头衔。
- 将岗位评估结果用于职级和薪酬带，不直接决定个人工资。
