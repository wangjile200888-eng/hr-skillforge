# Evidence Scoring

Every source card receives a score and evidence level.

## Source Tier Base Weights

| source_tier | Weight |
|---|---:|
| `official_video` | 1.3 |
| `public_interview` | 1.2 |
| `official_report` | 1.2 |
| `user_provided` | 1.3 |
| `official_article` | 1.0 |
| `case_study` | 1.0 |
| `academic` | 0.9 |
| `third_party_analysis` | 0.8 |
| `news` | 0.4 |
| `secondary_summary` | 0.2 |

## Add Points

| Condition | Points |
|---|---:|
| Official source | +30 |
| First-hand source | +25 |
| Long article / PDF / video transcript | +20 |
| Recent 3 years | +10 |
| Clear author/date | +10 |
| Contains case or tool | +10 |
| Strong HR/organization relevance | +10 |
| Official webinar/report/white paper | +15 |

## Deduct Points

| Condition | Points |
|---|---:|
| Secondary summary | -15 |
| No date | -10 |
| Obvious marketing with no substance | -20 |
| Duplicate content | -30 |
| Unclear source | -40 |
| Slogan only | -20 |
| Weak relation to target | -20 |

## Evidence Level

| Level | Score | Use |
|---|---:|---|
| A | 80+ | Can support core methodology |
| B | 60-79 | Supporting evidence |
| C | 40-59 | Background only |
| D | <40 | Do not enter final Skill |

Core claims require at least one A/B source, or multiple C sources clearly marked as weak signals.
